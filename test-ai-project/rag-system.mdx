## What it does

This system answers questions by searching your documents and passing the results to an LLM.

## My stack

- **Vector DB:** ChromaDB
- **LLM:** Claude
- **Framework:** LangChain

## How to run it

```python
from rag import query
response = query("your question here")
print(response)
```

## Things I always forget

- Re-embed documents if you change the embedding model
- Chunk size is set to 512 tokens
